# Prodesk IT Landing Page

A modern, responsive landing page for **Prodesk IT** built using **HTML, CSS, and JavaScript**. The website showcases the company's digital services with a clean user interface, responsive design, and dark mode support.



## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
